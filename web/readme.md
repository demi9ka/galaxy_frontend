# Как запустить
1. Устанавливаем yarn:

```shell
npm install -g yarn
```

2. Устанавливаем зависимости:

```shell
yarn install
```

3. Запускаем проект:

```shell
yarn run vite
```

- страница проекта расположена в `http://127.0.0.1:5173/`

## Важные файлы

- `src/App.tsx` - основная логика и тп.
- `src/styles/index.css` - css тема всякая.
- `src/main.tsx` - импорты всякие.
